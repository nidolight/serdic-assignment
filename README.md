## 📌 1. 프로젝트 개요 및 목표

이 프로젝트는 웹 환경에서 **3D 뎁스/포인트클라우드 데이터**를 효율적으로 시각화하고 탐색할 수 있는 뷰어 프로토타입을 구현하는 것을 목표로 합니다.

**주요 과제 주제:** 웹 기반 3D 뷰어 페이지 및 뎁스/포인트클라우드 뷰어 프로토타입 구현

**주요 목표:**

* 사용자 친화적인 페이지 레이아웃 및 웹 디자인 구현.
* 뎁스 데이터 및 포인트 클라우드 데이터를 웹에서 시각화하는 프로토타입 기능 검증.

## 🛠️ 2. 개발 환경 및 기술 스택 (Tech Stack) 🚀

이 프로젝트는 **웹 기반 3D 시각화**에 초점을 맞추어 아래와 같은 기술 스택을 사용합니다.

| 분류 | 기술 | 설명 |
| :--- | :--- | :--- |
| **개발 환경** | **Visual Studio Code** (VS Code) | 코드 편집 및 개발 환경 |
| **프론트엔드** | **React** (with **Vite**) | 빠르고 효율적인 UI 구축을 위한 라이브러리 및 번들러 |
| **3D 시각화** | **React Three Fiber** (**R3F**) | Three.js를 React 컴포넌트처럼 사용할 수 있게 해주는 선언적 바인딩 |
| **패키지 관리** | **npm** | Node.js 기반 프로젝트의 의존성 및 스크립트 관리 |

## 🚀 3. 빌드 및 실행 방법 요약 (Quick Start Guide)

프로젝트를 로컬 환경에서 실행하기 위한 간단한 가이드입니다.

### 3.1. 전제 조건

* **Node.js** (LTS 버전 권장) 및 **npm** 설치

### 3.2. 프로젝트 설치

1.  **레포지토리 클론:**
    ```bash
    git clone https://github.com/nidolight/serdic-assignment/
    cd serdic-assignment
    ```
2.  **의존성 설치:**
    ```bash
    npm install
    ```

### 3.3. 실행 및 빌드

| 명령어 | 설명 |
| :--- | :--- |
| `npm run dev` | 개발 모드로 서버를 실행합니다. (일반적으로 `http://localhost:5173` 접속) |
| `npm run build` | 배포를 위한 프로덕션용 파일(`dist` 폴더)을 빌드합니다. |

## 📐 4. 구현 내용 및 주요 기능 ✨

### 4.1. 웹페이지 디자인 및 퍼블리싱
* **페이지 레이아웃 설계:** **Figma** 등 디자인 툴을 활용하여 페이지 레이아웃 설계.
* **웹 퍼블리싱:** HTML/CSS/JS 등을 사용하여 사용자 친화적인 웹 페이지 구현.
* **디자인 유사성:** 레퍼런스 영상의 디자인과 **유사하게** 페이지 구성.

### 4.2. 3D 포인트 클라우드 뷰어 프로토타입
* **데이터 시각화:** 브라우저 상에서 **포인트 클라우드 데이터**를 로드하고 **Three.js / R3F**를 활용하여 3D 환경에 시각화.
* **기본 조작 기능:**
    * **회전:** 마우스 드래그를 이용한 뷰어 회전.
    * **이동 (Pan):** 뷰어 내 객체 이동.
    * **확대·축소 (Zoom):** 마우스 휠을 이용한 줌 인/아웃.

## ⚠️ 5. 미구현 및 향후 개선 방향 🚧

### 5.1. 미구현 사항
* **뎁스 맵 이미지 로드/시각화:** 현재 파일 로딩 및 디코딩 이슈로 **뎁스 맵 이미지를 직접 로드하여 시각화하는 기능**은 미구현.

### 5.2. 향후 개선 방향
* **데이터 파싱 및 디코딩:** `.npz` 파일 등 원본 데이터 포맷을 직접 분석하여 뎁스 정보의 **적절한 디코딩 및 파싱** 로직 구현.
* **UI/UX 컴포넌트화:** 현재의 UI 요소를 **React 컴포넌트** 단위로 분리하여 **재사용성 및 유지보수성** 향상.

## 🤝 6. 커밋 컨벤션 (Commit Convention)

이 프로젝트는 **Conventional Commits** 규칙을 따르며, 모든 커밋 메시지는 다음 형식 중 하나를 사용해야 합니다.

**형식:** `<타입>[선택적 범위]: <제목>`

| 타입 (Type) | 설명 (Description) | 예시 (Example) |
| :--- | :--- | :--- |
| **`feat`** | **새로운 기능** 추가 | `feat: 사용자 로그인 기능 구현` |
| **`fix`** | 버그 수정 | `fix: 3D 뷰어 초기 로드 시 깜빡임 현상 수정` |
| **`docs`** | 문서 변경 (README, 가이드 등) | `docs: 커밋 컨벤션 정보 추가` |
| **`style`** | 코드 스타일 변경 (포맷팅, 세미콜론 등, 로직 변경 없음) | `style: 함수 코드 포맷팅 정리` |
| **`refactor`** | 코드 리팩토링 (기능 변경 없이 코드 구조 개선) | `refactor: 뷰어 컴포넌트 로직 분리` |
| **`test`** | 테스트 코드 추가/수정 | `test: 3D 로더 유닛 테스트 추가` |
| **`chore`** | 빌드 시스템, 라이브러리 설치 등 기타 변경 (개발 환경 관련) | `chore: Three.js 라이브러리 업데이트` |
